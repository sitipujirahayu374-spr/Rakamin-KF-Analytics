## 📌 Deskripsi Proyek
Proyek ini merupakan bagian dari program *Project Based Internship* (PBI) yang diselenggarakan oleh **Rakamin Academy** bekerja sama dengan **Kimia Farma**. Sebagai seorang *Data Analyst Intern*, saya bertanggung jawab untuk menganalisis data transaksi, produk, dan cabang Kimia Farma untuk memberikan wawasan bisnis yang mendalam mengenai performa perusahaan.

Tujuan utama dari proyek ini adalah mengolah data mentah menggunakan **Google BigQuery** (SQL) dan menyajikannya dalam bentuk tabel analisa yang siap digunakan untuk keperluan visualisasi data atau pengambilan keputusan bisnis.

## 🛠️ Alat & Teknologi
* **Database Management:** Google BigQuery
* **Query Language:** SQL (Standard SQL)
* **Platform:** GitHub (Version Control)

## 📊 Dataset yang Digunakan
Dataset terdiri dari beberapa tabel utama:
1.  `kf_final_transaction`: Data transaksi penjualan produk.
2.  `kf_inventory`: Data stok barang di setiap cabang.
3.  `kf_kantor_cabang`: Detail informasi mengenai lokasi cabang.
4.  `kf_product`: Detail informasi mengenai produk dan harga.

## 🚀 Analisis yang Dilakukan
Dalam proyek ini, saya membuat sebuah **Table Analisa** yang menggabungkan informasi penting dari berbagai tabel di atas. Beberapa poin utama yang dihitung meliputi:
* **Gross Profit:** Menghitung persentase laba berdasarkan harga produk.
* **Nett Sales:** Total pendapatan setelah dikurangi diskon.
* **Nett Profit:** Laba bersih yang dihasilkan dari setiap transaksi.
* **Performance Rating:** Menggabungkan rating transaksi untuk melihat kepuasan pelanggan di tiap cabang.
